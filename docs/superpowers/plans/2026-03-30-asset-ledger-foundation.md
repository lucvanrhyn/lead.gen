# Asset Delivery And Operator Ledger Foundation Plan

**Goal:** Add hosted lead-magnet pages with tracked asset views and expand Google Sheets syncing into a multi-tab operator ledger.

**Scope for this slice:**
- one hosted lead-magnet asset per outreach draft
- asset page view tracking and follow-up generation on first high-intent view
- asset link included in Gmail draft delivery
- multi-tab Google Sheets sync for `Companies`, `Contacts`, `Drafts`, `LinkedIn Tasks`, and `Engagement`
- UI surfacing of asset links in the lead detail view

**Non-goals for this slice:**
- HubSpot mirroring
- live email open/click webhooks
- booking integrations
