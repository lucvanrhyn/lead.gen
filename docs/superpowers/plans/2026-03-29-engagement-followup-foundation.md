# Engagement And Follow-Up Foundation Plan

**Goal:** Add a first-class engagement timeline and automatic follow-up draft generation so the app can react to opens, clicks, asset views, and replies while keeping send approval human-controlled.

**Scope for this slice:**
- engagement event schema and persistence
- outreach draft sequencing with parent/child follow-up drafts
- event logging route for approved/sent-ready drafts
- automatic first follow-up generation from engagement signals
- lead detail engagement tab and manual signal logging actions
- seed data and tests

**Non-goals for this slice:**
- live Gmail reply sync
- live HubSpot mirroring
- hosted lead-magnet page tracking
- auto-send behavior
